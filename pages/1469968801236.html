CLIPBLOC CONCEPT

Imagine you have a notebook ( BLOC )...

Each page ( CLIP ) of your notebook can content text, images, functionalities...

But in this case you can cut, transfer, import... ( MOVE ) each page ( CLIP ) to the place you need...

And you can associate a page ( CLIP ) with a group of pages ( CLIPS )...

But the most important... you can create as many BLOCS you wants, and every BLOC would able to read contents from other BLOCS your or from others.

CONCEPTO DE CLIPBLOC

Imagina un cuaderno de notas ( BLOC )...

Cada página ( CLIP ) de tu cuaderno puede contener textos, imágenes, funcionalidades...

Pero en este caso podrás cortar esas hojas, importarlas, transferirlas...( MOVERLAS ), cada hoja ( CLIP ) donde la quieras colocar o necesites...

Puedes asociar una hoja ( CLIP ) con grupos de hojas ( CLIPS )...

Pero lo más importante podrás crear tantos BLOCS como necesites, y en cada BLOCpodrás leer contenidos de otros BLOCS tuyos o de otras personas.
